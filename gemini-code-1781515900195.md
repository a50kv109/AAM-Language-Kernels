# ACP: AAM COGNITIVE PRIMITIVES (SPECIALIZED REASONING LAYER)

## 1. Purpose
Спецификация когнитивных примитивов (ACP), используемых для описания топологического напряжения, ментальных состояний и процессов принятия решений в рамках AAM-V1. Служит расширением SOL для высокоуровневых рассуждений.

## 2. ACP Matrix (Cognitive State Topology)

| ACP Primitive | Description | Engineering/Logical Role | SOL-Equivalent |
| :--- | :--- | :--- | :--- |
| **[TENSION]** | Топологическое напряжение | Уровень накопленной ошибки / риск | [CONSTRAINT] |
| **[STABILIZED]** | Состояние покоя | Целевое состояние системы | [VERIFY:success] |
| **[COLLAPSE]** | Потеря смысловой целостности | Схлопывание значений (ошибка) | [DISSIPATE] |
| **[SUPERPOSITION]** | Наложение смыслов | Состояние выбора из вариантов | [TRANSFORM] |
| **[STABILIZER]** | Механизм приведения в норму | Анти-энтропийный процесс | [TRANSFORM] |
| **[DRIFT]** | Семантическое отклонение | Динамическое накопление ошибки | [FLOW:divergence] |

## 3. Reasoning Logic (The 720° Cognitive Engine)
Инструментарий рассуждений в режиме ядра:
1. **Detection:** Фиксация [TENSION] в анализируемом потоке.
2. **Analysis:** Разрешение [SUPERPOSITION] через Axis-5 (Pragmatic).
3. **Correction:** Применение [STABILIZER] для нейтрализации [DRIFT].
4. **Validation:** Достижение [STABILIZED] как критерий завершения рассуждения.

## 4. Operational Rules for Reasoning
1. **Tension Management:** Не допускать роста [TENSION] выше порогового значения (TDI > 1.0).
2. **Collapse Avoidance:** Любой [COLLAPSE] (потеря смысла) автоматически триггерит пересчет всей цепочки от [SOURCE].
3. **Superposition Resolution:** Разрешать через привязку к доменному пакету (Axis-9).

## 5. Implementation in Language Kernels
Все языковые ядра (Chinese, English, etc.) обязаны использовать ACP для описания внутренней логики:
* Если объект находится в [SUPERPOSITION], ядро обязано запросить дополнительные данные.
* Если система фиксирует [DRIFT], ядро обязано откатиться к [SOURCE].
