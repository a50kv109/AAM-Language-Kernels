# CHINESE CORE: METAL STATE & COLOR SEMANTICS

## 1. Scope
Слой семантической стабилизации для инженерной диагностики материалов по их визуальным признакам (окисление, коррозия, пассивация).

## 3. Semantic Rules for Engineering Diagnostics
Для исключения ошибок интерпретации состояния материала при анализе текста:

1. **Rule 1 (Protection vs Corrosion):** 
   - Если признак "защитный" (например, патина, оксидная пленка), система маппит это на [CONSTRAINT:protection], а не на разрушение.
2. **Rule 2 (Color Change Origin):** 
   - Различать окисление от кислорода (氧化) и от сульфидов (硫化). Например, почернение серебра не является коррозией в классическом смысле разрушения (не требует замены детали).
3. **Rule 3 (Optical Effects):** 
   - Цвета титана классифицируются как [TRANSFORM:anodize] (анодирование), а не как внешнее покрытие (краска/油漆).

## 4. Engineering Context Map
* **[STATE:normal]** → 正常状态 (Zhèngcháng zhuàngtài)
