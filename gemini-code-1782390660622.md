# Semantic Normalizer Engine
## System Key: AAM-V1_ARTSYBASHEV_UA_KHARKIV_AIANALYSIS
**Status: SPECIFICATION**

Семантический нормализатор — это компилятор, очищающий лексический вход от языкового шума и собирающий структуру EIR.

* **API (Вход):** `Parse_to_EIR(language_code, text_input)`
* **API (Выход):** `Generate_from_EIR(language_code, EIR_object)`
* **Правило:** Разные слова в разных языках, описывающие один инженерный объект, должны сводиться к одному примитиву Axis-8/EIR.
