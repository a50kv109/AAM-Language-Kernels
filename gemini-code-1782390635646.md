# Engineering Intermediate Representation (EIR)
## System Key: AAM-V1_ARTSYBASHEV_UA_KHARKIV_AIANALYSIS
**Status: SPECIFICATION**

EIR — это внутренний машинный язык репозитория, наследуемый от примитивов SOL.
Языковые ядра должны конвертировать текст в сущности EIR:

* `EIR_Entity`: Базовый класс физического объекта (Вал, Отверстие, Документ).
* `EIR_Constraint`: Физическое/геометрическое ограничение (Допуск, Натяг, Зазор). Оперирует числовыми границами.
* `EIR_Flow`: Вектор передачи энергии/силы.
* `EIR_Topology`: Пространственные отношения (вложено, параллельно).
